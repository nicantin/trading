# Iran Conflict — Trading Dashboard

Dashboard interactif de suivi du conflit Iran 2026 pour le momentum trading.

## Secteurs suivis
- ⚓ Tankers / Shipping
- 🛢️ O&G Canadien
- 🥇 Or / Gold
- 🌾 Fertilisants / Potash
- 🏗️ Gulf Short (Dubai/UAE)

## Features
- Score composite d'escalation (1-10)
- KPIs par secteur avec tickers individuels
- Scénarios Bull/Base/Bear avec price targets
- Matrice de dé-escalation interactive (clic pour toggle)
- Mini bar charts de tendance
- Sources gratuites et routine quotidienne

## Déploiement

Le site se déploie automatiquement via GitHub Pages à chaque push sur `main`.

URL: `https://<ton-username>.github.io/iran-dashboard/`

## Setup local (optionnel)

```bash
npm install
npm run dev
```
